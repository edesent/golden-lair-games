# Golden Lair Games

Modern Next.js redesign for Golden Lair Games, preserving the original black and gold brand, logo, store imagery, schedule flyer, location, hours, and community/store messaging.

## Local Development

```bash
npm install
npm run dev
```

## Production

```bash
npm run build
```
